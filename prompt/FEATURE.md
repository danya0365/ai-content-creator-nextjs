ผมยกให้คุณทำหน้าที่เป็นคอนเทนต์ครีเอเตอร์ หารายได้จาก content โดยให้ generate รูปภาพ pixel art แล้วเก็บลงใน database 

เช่น เนื้อหาตลกๆ และ generate เนื้อหานั้นเป็นรูป pixel art 

คุณช่วยวางแผนงานให้ผมหน่อยครับ ในแต่ช่วงเวลา ในแต่ละวัน ในแต่ละสัปดาห์ ควรโพสส์คอนเทนต์แบบไหนให้ต่อเนื่อง เช่น ตอนเช้า โพสท์ คอนเทนต์สรุปข่าวเช้า พร้อมรูป pixel art น่ารักๆ ตอนเที่ยงโพสท์ คอนเทนต์เกี่ยวกับอาหาร generate pixel art รูปอาหาร อะไรทำลองนี้ ช่วยวางแผนหน่อยคับ

การ generate รูปภาพ pixel art ต้องใช้ Google Gemini Banana API (ผมมี key เสียเงิน)

หลักๆคือ ผมต้องการให้คุณช่วยเขียนโค้ด เมื่อ generate รุปตาม ตามช่วงเวลา โดยใช้ cronjob ของ nextjs
